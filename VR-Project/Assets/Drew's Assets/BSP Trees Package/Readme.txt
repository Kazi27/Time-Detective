To customize the wind effect select any material (bark or foliage) and find 'Wind Settings'. You can set parameters such as: wind direction, strength, speed, turbulence + additional flutter effects for foliage.

To install URP or HDRP shaders and materials select '(URP) Trees Package' or '(HDRP) Trees Package' package file that you will find in the main folder and import all files.
If the materials are still pink, close the Editor and open it again.
